# 游戏引擎中的粒子和声效系统

## 粒子基础

Particle:
A particle in game is usually a sprite or 3D model, with following attributes:
- Position
- Velocity
- Size
- Color
- Lifetime


Particle Emitter:
- Specify spawn rules
- Specify simulation logic
- Describe how to render particles

Particle System:
A collection of individual emitters

粒子生成行为模型：

不同发射模式：
- Continuous
- Burst

Simulate

Particle Type:
- Billboard Particle
- Mesh Particle
- Ribbon Particle
